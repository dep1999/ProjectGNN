📊 字段说明：
- gene：基因的 HGNC symbol（标准名）
- RNAseq_LFC：KRAS G12C 阳性 vs 非阳性样本之间的 log2 fold changelog2 fold change（log2FC） 是一种用对数比例（以2为底）表示差异倍数的方式。其计算公式如下：
  log2FC=log2(下标)(处理组数值/对照组数值)log2FC > 0 表示上调；log2FC < 0 表示下调；log2FC = 0 表示无变化
- RNAseq_pval：上述比较的 t 检验 P 值
- RNAseq_mean：该基因在所有样本中的平均表达量
- RNAseq_std：该基因表达量的标准差（变异性）

🧠 如何用于推荐系统？

该表达特征表中的字段将用于构建推荐系统特征向量矩阵 app_data_kras.csv 中的以下维度：
- RNAseq_LFC：用于评估该基因在 KRAS G12C 阳性细胞中的差异表达强度（目标方向 = high）
- RNAseq_pval：用于提供表达差异的置信度（可作为辅助筛选或加权因子）
- RNAseq_mean + RNAseq_std：用于衡量该基因整体表达水平与变异性（有助于剔除本底表达极低或高度不稳定的基因）

这些特征将与 CRISPR KO 得分、图谱距离、文献共现频率等特征一起输入至 SkywalkR 多目标推荐系统中，以帮助优先识别出与 KRAS G12C 耐药机制相关的潜在驱动因子。
